# Hero Rabbit Chrome Extension

The goal of this extension is to bring to the RabbitMQ Management Plugin interface features that are very useful for software engineers and SREs who deal with RabbitMQ clusters on a daily basis.
These features can be considered `unconventional`, but during my brief career in the software industry, they have saved my day multiple times.

## Quick Start

##### If your organization allows it, simply download the extension from the Chrome Store.

[![Baixar na Chrome Web Store](https://upload.wikimedia.org/wikipedia/commons/e/e1/Google_Chrome_icon_%28February_2022%29.svg)](https://chromewebstore.google.com/detail/boxel-3d/mjjgmlmpeaikcaajghilhnioimmaibon)

##### Or your can following this steps to install manually:

<Steps>
    ### Get the last release

[Latest Release](https://github.com/vanascimento/herorabbit/releases/latest)

    ### Get the source code and select the dist folder

    ### Upload the dist folder to your Chrome Extensions in Developer Mode

    ### Enjoy

</Steps>
