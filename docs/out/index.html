Getting started

Hero Rabbit Chrome Extension

The goal of this extension is to bring to the RabbitMQ Management Plugin interface features that are very useful for software engineers and SREs who deal with RabbitMQ clusters on a daily basis. These features can be considered unconventional, but during my brief career in the software industry, they have saved my day multiple times.

Quick Start
